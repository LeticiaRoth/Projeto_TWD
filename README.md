# TWD Quest

> *"Nem a tecnologia mais avançada pode substituir a coragem de um sobrevivente determinado."*

Uma aplicação gamificada que transforma a escola em uma zona de sobrevivência. Explore os corredores como se fossem o apocalipse zumbi, responda enigmas e libere os grandes sobreviventes para completar a **Ordem de TWD**!

---

## Sobre o Projeto

**Projeto TWD** é um Caça-Tesouros interativo desenvolvido como atividade somativa para a unidade curricular de **Programação Web Front-end 2** no SENAI "Roberto Mange". A proposta é transformar o ambiente físico da escola em um mapa pós-apocalíptico navegável, onde os alunos percorrem pontos reais e desbloqueiam personagens do universo **The Walking Dead** ao responderem quizzes e resolverem enigmas.

O objetivo final é reunir todos os **6 personagens** e completar a Ordem de TWD.

---

## Personagens e Pontos de Encontro

### 🧒 Carl Grimes
**Local:** Sala dos Professores  
**Conexão:** A sala dos professores traz um ar de aprendizado. Carl precisou crescer e aprender a sobreviver muito rápido, sendo sempre um exemplo para os outros.  
**Frase Marcante:** *"Meu pai me ensinou a sobreviver. Mas sobreviver não é suficiente."*

---

### 🍕 Glenn Rhee
**Local:** Lanchonete  
**Conexão:** Antes do apocalipse, Glenn trabalhava como entregador de pizza. A lanchonete simboliza sua vida antes do caos.  
**Frase Marcante:** *"Eu irei encontrar você!"*

---

### 🏹 Daryl Dixon
**Local:** Oficina  
**Conexão:** A oficina representa conserto, armas e veículos. Daryl é especialista em sobrevivência, caça e manutenção de equipamentos.  
**Frase Marcante:** *"Nós não somos como eles."*

---

### 👮 Rick Grimes
**Local:** Sala do Lucas  
**Conexão:** A sala simboliza liderança e decisões difíceis. Rick sempre foi o líder que precisava orientar, proteger e tomar decisões duras pelo grupo.  
**Frase Marcante:** *"Nós somos aqueles que vivem."*

---

### ⚔️ Michonne
**Local:** Busto Mange  
**Conexão:** Uma estátua representa força, resistência e postura firme. Michonne é símbolo de resistência.  
**Frase Marcante:** *"Eu não tenho mais medo."*

---

### 🚨 Negan
**Local:** Biblioteca  
**Conexão:** A biblioteca representa estratégia, planejamento e poder psicológico. Negan liderava através do medo e da manipulação.  
**Frase Marcante:** *"Eu sou Negan."*

## Como Rodar o Projeto

### Pré-requisitos

- [Node.js](https://nodejs.org/) v18 ou superior
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

### Instalação

```bash
# 1. Clone o repositório
git clone https://github.com/LeticiaRoth/Projeto_TWD.git

# 2. Entre na pasta do projeto
cd projetoTWD

# 3. Instale as dependências
npm install
```

### Iniciando o projeto

```bash
npm run dev
```

O app abrirá em `http://localhost:5173/`.
